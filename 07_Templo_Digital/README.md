# 🏛️ Templo Digital

> *Experiencias interactivas, rituales digitales y portales visuales*

---

## 🎯 ¿Qué es esto?

El **Templo Digital** es el espacio sagrado donde la tecnología se vuelve místico-operativa.

Aquí convergen:
- **HTML interactivo** - Experiencias visuales contemplativas
- **Animaciones hipnóticas** - Objetos de meditación digital
- **Rituales codificados** - Ceremonias en JavaScript
- **Portales navegables** - Experiencias inmersivas

---

## 🌟 Características

### **Sin Dependencias**
Todo el código es autónomo. HTML, CSS y JavaScript puro.

### **Diseño Contemplativo**
Optimizado para sesiones largas sin cansar la vista.

### **Responsive**
Funciona en desktop, tablet y móvil.

### **Sagrado + Técnico**
Cada línea de código es una oración. Cada función es un ritual.

## 💫 Contenido

### **Nodos Visuales Interactivos**
Los 91 nodos de Trinity Infinito desplegados como experiencias web.

### **Meditaciones Guiadas**
Rituales digitales con audio, visual y texto sincronizados.

### **Simuladores Dimensionales**

## 🔧 Tecnologías

- **HTML5** semántico
- **CSS3** con animaciones nativas
- **JavaScript vanilla** (sin frameworks)
- **Canvas API** para gráficos
- **Web Audio API** para sonido
- **Three.js** para 3D (cuando necesario)
- **Tone.js** para síntesis de audio
